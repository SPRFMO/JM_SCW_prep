# Some evaluations of revised age data

This folder is meant as an approach to evaluate aspects of the new age data provided by Chile for fleets S-C Chile and and North and also the survey

| File      | Description |
| ----------- | ----------- |
| `Age_EDA.Rmd` | notebook for explorations (main R code) |
| `Age_EDA.html` | knitted version of notebook for explorations (view of results) |
| `AgeDataInAssessment.csv` | From the last assessment input file from 2020 |
| `data.Rproj` | Rstudio project file |
| `All_Age_Data_Chile_Pivot.xlsx` | Some excel view |
| `All_Age_Data_Chile.xlsx` | Revised Chilean data w/ field explanations |
| `All_Age_Data_Chile.csv` | Revised Chilean data in one file for reading into R |
